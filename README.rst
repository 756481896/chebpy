Installation
------------

To install ChebPy, you should first install the `fftw` library::

    # On Linux
    $ sudo apt-get install libfftw3-dev

    # Via Homebrew on Mac
    $ brew install fftw

And then proceed to install chebpy with::

    $ python setup.py install
